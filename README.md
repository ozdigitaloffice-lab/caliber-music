# משפחת קליבר · Caliber Family

האתר הרשמי של להקת ההיפ-הופ **משפחת קליבר**. אסתטיקת *Kinetic Brutalism* — שחור עמוק + אסיד-יילו, אנימציות מסע סקרול, ו-marquees אינסופיים.

## 🎯 מה האתר עושה

מאגר 17 הסינגלים של הלהקה — לכל שיר עטיפה אמיתית מ-Apple Music + 3 כפתורי השמעה (Spotify, Apple Music, YouTube).

## 🛠 Stack

- **Next.js 16** (App Router, Turbopack)
- **React 19**
- **TypeScript**
- **Tailwind CSS v4** (CSS-first config)
- **Framer Motion** — interactions, modals, 3D tilt
- **GSAP + ScrollTrigger** — sticky hero with scroll-driven video transforms
- **Lenis** — smooth scroll
- **next/font** — self-hosted Heebo + Suez One + Space Grotesk + JetBrains Mono
- **next/image** — optimized album artwork from Apple's CDN

## 📁 Structure

```
src/
├── app/
│   ├── layout.tsx        ← RTL (dir="rtl"), fonts, body wrappers
│   ├── globals.css       ← Tailwind v4 @theme tokens (Kinetic Brutalism)
│   └── page.tsx          ← Composes the homepage sections
├── components/
│   ├── SmoothScroll.tsx  ← Lenis init (respects prefers-reduced-motion)
│   ├── Grain.tsx         ← Persistent SVG noise overlay
│   ├── Marquee.tsx       ← CSS-only infinite scroll
│   ├── Hero.tsx          ← Sticky hero with GSAP-scrubbed YouTube embed
│   ├── SongGrid.tsx      ← Bento layout
│   ├── SongCard.tsx      ← 3D tilt-on-hover card
│   ├── PlatformPicker.tsx← Modal with 2-3 platform buttons
│   ├── PlatformIcons.tsx ← Inline SVG brand marks
│   ├── AboutStrip.tsx    ← Big-numbers stat band
│   └── Footer.tsx        ← Bottom band + social links
├── lib/songs.ts          ← Typed accessor over data/songs.json
└── data/songs.json       ← The 17 songs catalog

_data/                    ← Raw scrapes + python script that builds songs.json
└── deep_search.py        ← Re-run when songs change

design-system/caliber-family/
├── MASTER.md             ← Auto-generated by ui-ux-pro-max skill
└── pages/home.md         ← Project-specific Kinetic Brutalism override
```

## 🚀 Development

```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # Production build
```

## 🌐 Deployment

Push to `main` → Vercel auto-deploys.

First-time Vercel setup:
1. https://vercel.com/new
2. Import `orizur123/caliber-FAMILY`
3. Accept defaults (Next.js auto-detected)

Album artwork CDN hostnames are already allow-listed in `next.config.ts`.

## 🔁 Refreshing the song catalog

```bash
python _data/deep_search.py
```

Uses iTunes Search API + yt-dlp + Spotify embed scraping. All unauthenticated.
